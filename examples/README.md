# Sample Cases

These files are checked-in sample examples.

- They mirror the API schema of the current service.
- They are written to reflect the zero-cost `mock` mode path.
- They are not meant to replace fresh benchmark or test runs.

Files:

- `reference_agency_meeting_assistant.json`
- `reference_local_repair_marketplace.json`
- `reference_clinic_compliance_assistant.json`
- `reference_meal_planning_subscription.json`

To regenerate fresh runtime outputs, run the API locally and call the endpoints or run `make benchmark` / `make benchmark-local`.
