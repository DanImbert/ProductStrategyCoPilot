# Generated Artifacts

This folder contains checked-in and generated artifacts from benchmark and regression runs.

- `reference_benchmark_results.csv` mirrors the CSV schema written by `scripts/benchmark.py`
- `reference_benchmark_summary.md` mirrors the Markdown summary shape written by `scripts/benchmark.py`
- `prompt_regression_results.json` is generated by `scripts/prompt_regression.py`
- `prompt_regression_summary.md` is generated by `scripts/prompt_regression.py`

The checked-in mock-mode files are representative samples and are not guaranteed to match a fresh run byte-for-byte.

To generate current artifacts locally:

```bash
make benchmark
make benchmark-local
make regression
```
